# Project 2: The Blueprint (Wireframing)

## Overview

This project focuses on creating a low-fidelity wireframe for an e-commerce homepage. The goal is to improve user flow, information hierarchy, and navigation by designing a clean and structured layout.

## Problem Statement

An e-commerce platform is losing customers during the checkout process due to a cluttered interface and confusing layout. A simplified homepage structure is needed to improve usability and guide users effectively.

## Objective

To design a low-fidelity homepage wireframe that prioritizes clarity, navigation, and clear call-to-action elements.

## Deliverables

* Homepage Wireframe
* Navigation Structure
* Information Hierarchy
* Call-to-Action (CTA) Placement
* User Flow Design

## Key Learnings

* Wireframing Fundamentals
* Information Architecture
* User Flow Design
* Visual Hierarchy
* Low-Fidelity Prototyping
